# Highrise Bot - بوت هايرايز

## نظرة عامة
بوت متقدم للعبة Highrise يوفر العديد من الميزات المتطورة للإدارة والترفيه.

## الميزات الرئيسية

### نظام الترجمة 🌐
- ترجمة النصوص تلقائياً: `ترجمة [النص]`
- يدعم جميع اللغات:
    - إذا كتبت بالعربي -> يترجم للإنجليزي.
    - إذا كتبت بأي لغة أخرى -> يترجم للعربي.
- مثال: `ترجمة hello` -> "مرحبا"
- مثال: `ترجمة مرحبا` -> "Hello"

### نظام الطوابق السريع ⚡
- **انتقال سريع للطوابق**: اكتب "طلعني" متبوعة برقم الطابق (1-3)
- **أوامر الطوابق**:
  - `طلعني 1` - الطابق الأول
  - `طلعني 2` - الطابق الثاني
  - `طلعني 3` - الطابق الثالث
  - `floors` أو `طوابق` - عرض قائمة الطوابق المتاحة

### نظام الرقصات والحركات
- رقصات متنوعة مع أوامر سهلة
- نظام loop للرقص المستمر
- أوامر رقص جماعي للمشرفين
- رقصات خاصة: ghost, rest, floss forever, dance floor

### نظام النقاط والمستويات
- كسب نقاط تلقائي عند الكتابة في الروم
- نظام مستويات متقدم (0-16)
- عرض النقاط: `نقاطي`

### نظام الإدارة
- إدارة المشرفين والأدمن
- نظام VIP متقدم
- حماية المستخدمين المميزين
- نظام السجن المؤقت
- أوامر النقل والتحكم

### ميزات إضافية
- نظام التنقل التلقائي على النقر
- حفظ المواقع المخصصة
- نظام الترحيبات الخاصة
- أوامر الـ spam
- فحص السرعة (ping)
- عرض مدة تشغيل البوت (time)

## الأوامر الأساسية

### أوامر الطوابق (جديد!)
- `طلعني 1` - `طلعني 3`: الانتقال السريع للطابق
- `floors`, `طوابق`: عرض قائمة الطوابق

### أوامر عامة
- `list` - قائمة الرقصات
- `ping` - فحص سرعة الاتصال
- `time` - مدة تشغيل البوت
- `نقاطي` - عرض نقاطك ومستواك

### أوامر الرقص
- `[رقم]` - تنفيذ رقصة برقمها
- `loop [رقم]` - رقصة مستمرة
- `0` أو `stop` - إيقاف الرقص
- `ghost`, `rest`, `floss forever`, `dance floor` - رقصات خاصة

### أوامر المشرفين
- `اسحب @username` - جلب المستخدم
- `tel @username` - الانتقال للمستخدم
- `follow @username` - متابعة المستخدم
- `الكل vip` - نقل جميع المتواجدين لمنطقة الـ VIP (يجب حفظ موقع باسم vip أولاً)
- `all [رقم]` - رقصة لجميع المستخدمين
- `h all` - قلوب للجميع
- `loopall [رقم]` - رقص جماعي مستمر
- `stopall` - إيقاف الرقص للجميع

### أوامر الأونرز
- `addtele [اسم]` - حفظ موقع جديد
- `removetele [اسم]` - حذف موقع محفوظ
- `!addvip @username` - إضافة VIP
- `!removevip @username` - إزالة VIP
- `!mod @username` - إضافة أدمن
- `!delmod @username` - حذف أدمن
